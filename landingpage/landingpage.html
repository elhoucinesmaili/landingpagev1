About The Project

As an avid traveler and developer, I often found myself frustrated with the cluttered interfaces of existing booking platforms. During a trip to Bali where I struggled to find affordable last-minute accommodations, I realized there was room for improvement in how we discover and book hotels.

This Booking.com clone was built over one months as my ALX portfolio project, combining my passion for travel with my software development skills. I focused on creating a cleaner, more user-friendly interface while maintaining all the essential functionality.

This project demonstrates my full-stack capabilities, featuring a React frontend, Clone with NEXT.JS 14 (Scrape Data w/Oxylabs, Shadcn, Tailwind CSS)
